O que é JavaScript?

JavaScript é a linguagem que faz a interação acontecer nas páginas web. Sem ele, a página é só um documento estático.

JavaScript se conectando ao HTML e CSS para fazer a página mais dinâmica. Alterando textos, respondendo ao clique e até modificando o layout.
